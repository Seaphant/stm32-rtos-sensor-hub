Apply these ONLY after STM32CubeIDE has generated the project into this folder
(board NUCLEO-G474RE, project sensor-hub-g474, FreeRTOS CMSIS_V2, LPUART1 115200 8N1,
I2C1 PB8/PB9 ~100 kHz, PE8 GPIO output label LD2).

=== freertos.c ===
In /* USER CODE BEGIN Includes */:
#include "app.h"

In /* USER CODE BEGIN Init */ inside void MX_FREERTOS_Init(void):
  app_init_early();

In /* USER CODE BEGIN RTOS_THREADS */:
  app_rtos_create_tasks();

=== syscalls.c ===
In /* USER CODE BEGIN Includes */ (or first USER block):
#include "stm32g4xx_hal.h"
extern UART_HandleTypeDef hlpuart1;

Inside _write, /* USER CODE BEGIN _write */ only (parameter may be "file" or "fd"):
  if (file != 1 && file != 2) {
    return -1;
  }
  if (HAL_UART_Transmit(&hlpuart1, (const uint8_t *)ptr, (uint16_t)len, HAL_MAX_DELAY) != HAL_OK) {
    return -1;
  }
  return len;

=== main.c ===
No app_init_early() if it lives in freertos.c Init block only.
Optional: /* USER CODE BEGIN Includes */ -> #include "app.h"

=== CubeIDE: link folders (Properties -> Paths and Symbols) ===
Source: link App, Services, modules, Common (this repo tree).
Includes: ${ProjDirPath}/App, Services, modules, Common
